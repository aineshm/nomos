Nomos · RL Environment Architecture

Decentralized cars that coordinate
without traffic signals.

Each agent has its own origin, destination, and local perception — no communication, no hive mind. It optimizes travel time under hard safety constraints, judged by a deterministic verifier.

The four separations that define the design

Principle 01

Agency, no hive mind

Shared policy weights, but each car acts on its own local observation. No inter-agent comms or shared runtime state.

Principle 02

Simulation ≠ Rendering

Compute (physics + RL) produces poses; the render layer just draws them. Different engines, never conflated.

Principle 03

Two world models

A tiny in-loop predictor drives (µs). Cosmos 3 reasons & generates out of loop (seconds). Cosmos never steers.

Principle 04

Verify the trace

Reward & validity come from a deterministic geometric pass over the log — not a re-simulation, not an LLM.

The experiment → result → reward → fine-tune loop

01

Experiment

Roll out the current policy on a seeded world cohort (seed + scenario + checkpoint).

02

Result · Trace

Log every car's per-step pose + events (crash, off-road, arrival).

03

Verifier

Deterministic geometric pass: arrived? travel time? valid? + constraint costs.

04

Reward

−travel_time, subject to costs ≤ 0 (Constrained MDP).

05

Fine-tune

MAPPO update + Lagrangian λ step. (Policy update — not LLM fine-tuning.)

The abstract car & the objective

Abstract car (immutable record)

Car:
  id
  origin, destination         # per-agent context
  route: [waypoints]          # OSM A* / cached
  pose:       (x, y, z), heading
  kinematics: speed, steer, lane
  progress:   wp_ptr, dist_remaining, travel_time
  status:     ACTIVE | ARRIVED | CRASHED
  costs:      crash, off_road, rule   # constraints

Action: [accel/brake, steer, lane_change]

Constrained MDP

maximize  E[ reward ]
          reward = −travel_time (+ arrival)

subject to
  crashes         ≤ 0
  off_road        = never
  rule_violations ≤ 0

# crash = dense per-agent terminal COST,
# not a whole-run discard (else no gradient).
# "valid run" = an eval metric, not the signal.
